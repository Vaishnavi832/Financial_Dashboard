Project Overview:-
This project focuses on financial data analysis using Microsoft Power BI.
The Sales Data Analysis dashboard provides insights into sales performance, revenue trends, and key financial metrics to support data-driven business decisions.

Objectives:-
Analyze overall sales performance.
Track revenue trends over time.
Identify top-performing products and regions.
Support financial decision-making through visual insights.

Tools & Technologies:-
Power BI
Excel / CSV (Data Source

Dashboard Features:-
Total Sales & Revenue KPIs
Monthly and yearly sales trends
Region-wise and product-wise sales analysis
Interactive filters and slicers
Clear and user-friendly visualizations
